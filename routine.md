# Routine (LievitoMath)

Questa è la checklist “routine” da seguire quando facciamo un rilascio/aggiornamento.

- Verifica avvio devserver
- `+1` sulla versione (es. da `0.0.0` a `0.0.1`)
- Aggiornamento mese (`Apr`, `Mag`, `Giu`, ...)
- Generazione `.txt` sul Desktop con il **codice completo dell’app**
- Generazione `.zip` sul Desktop con backup completo dell’app
- Verifica `npm run build`
- `git commit` e `git push`
